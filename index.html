What is Difference between data science and machine learning

Data science = statistical analysis + data engineering in order to get information from the data

Machine learning = create a model to predict something. You can also define it as teaching computers to learn by examples
What is a random variable?
It assign a numerical value to each possible outcome. A numeric value is a necessary condition in the sense that something not numeric chosen at random is NOT a random variable. Remember that values of random variables are determined by chance
Can you explain the two types of random variable discrete and continuous?
Discrete random variables have a finite number of possible outcomes Continuous random variables you cannot count the number of possible outcomes
What is a probability distribution?
A probability distribution displays the probabilities associated with all possible outcomes.

Machine learning

How does machine learning differ from data mining?
Machine learning aims at predictions based on known properties learned from the training data. Data mining focuses on discovery of unknown properties on the data
